HTML Lists Cheat Sheet

List Type Tag Description Example
Ordered List <ol> A numbered list. 
<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>
Unordered List <ul> A bulleted list. 
<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>
Definition List <dl> A list of terms and definitions. 
<dl>
    <dt>Term</dt>
    <dd>Definition</dd>
</dl>
